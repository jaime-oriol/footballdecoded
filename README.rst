FootballDecoded
===============

Professional football data extraction and analysis toolkit for tactical analysis.

Overview
--------

FootballDecoded provides a comprehensive toolkit for extracting, processing, and visualizing football data from multiple sources including FBref, Understat, and WhoScored.

Features
--------

* **Data Extraction**: Automated scrapers for FBref, Understat, and WhoScored
* **Data Storage**: PostgreSQL-based persistence with quality scoring
* **Visualization**: Publication-quality charts for tactical analysis
* **Analysis Tools**: Advanced metrics including xG, pass networks, and spatial analysis

Installation
------------

.. code-block:: bash

    pip install -e .
